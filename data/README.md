# Data

## raw/

Observational data used in Paper 2:

- **DESI DR2 BAO**: DV/rd measurements across redshift bins (source: DESI Collaboration 2024).
- **Planck 2018**: Compressed prior (Ωm h², Ωb h², θ_s) from CMB TT+TE+EE+lowE.
- **Galaxy cluster masses**: Dynamical mass estimates used to test Ωm_eff = 0.160.

Data files are not committed to the repo due to size. See each script in `src/` for the exact values used (hardcoded from published tables).

## processed/

Derived inputs or pre-processed arrays generated by the pipeline. Produced by `src/ssee_paper2_analysis.py`.
